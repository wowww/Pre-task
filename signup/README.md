# signup


파일 다운로드 후,   
터미널 열고 현재 내 파일 위치 확인  
`cd 파일명`: 해당 파일명의 파일로 진입  
`../`: 뒤로 가기   
등의 터미널 명령어를 통해 해당 파일(signup) 경로로 이동해준다.  


```
npm install
```

```
npm run serve
```
